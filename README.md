# Portfolio

A modern React portfolio website with dark mode and responsive design.

## Setup

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Customization

Update your information in the component files:
- `src/components/Hero/Hero.js` - Name and intro
- `src/components/About/About.js` - About section
- `src/components/Experience/Experience.js` - Work history
- `src/components/Skills/Skills.js` - Skills and certifications
- `src/components/Projects/Projects.js` - Projects showcase
- `src/components/Contact/Contact.js` - Contact info

## Features

- Dark/light mode toggle
- Responsive design
- Contact form
- Project filtering
- Smooth animations
